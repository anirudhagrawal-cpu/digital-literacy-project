Project Overview

This repository contains my Digital Literacy Project created as part of the CSE0001 course. The purpose of this project is to understand the importance of digital literacy and to learn how to use digital tools in a safe and professional way. In this project, I have completed five tasks related to digital awareness, professional online presence, coding platforms, email etiquette, and cybercrime awareness.

All the files, screenshots, and documents related to each task are organized in separate folders inside this repository.

Task Summary
Task 1 – Digital Literacy Infographic

In this task, I created a digital literacy awareness infographic using Canva. The infographic explains what digital literacy is, safe internet practices, and useful digital tools for students.

Task 2 – Student Digital Portfolio

In this task, I created and updated my student digital portfolio on platforms such as GitHub and LinkedIn. This helped me understand how to build a professional online presence.

Task 3 – Coding Platform and Google Form

In this task, I created an account on a coding platform and completed a beginner-level problem. I also created a Google Form to check digital literacy awareness among students.

Task 4 – Email Etiquette and Social Media Awareness

In this task, I wrote two professional emails and created a social media Do’s and Don’ts checklist. This task helped me understand the importance of professional communication.

Task 5 – Cybercrime Awareness

In this task, I created a cybercrime case study and a prevention checklist. This helped me understand how cybercrime affects students and how to stay safe online.
